Índice de Masa Corporal (IMC)

¿Qué es el IMC?

El Índice de Masa Corporal (IMC) es una medida que evalúa la relación entre el peso y la altura de una persona. Se utiliza comúnmente para determinar si una persona tiene un peso saludable en relación con su altura.

Importancia del IMC

El IMC es una herramienta útil para identificar posibles problemas de peso que pueden estar relacionados con la salud. Puede indicar si una persona tiene bajo peso, peso normal, sobrepeso u obesidad, lo que puede ayudar a prevenir problemas de salud asociados con el exceso de peso.

Cómo se calcula el IMC

El IMC se calcula dividiendo el peso en kilogramos entre el cuadrado de la altura en metros.

Fórmula: IMC = Peso (kg) / (Altura (m) * Altura (m))

Valores para adultos

Los valores típicos del IMC para adultos son:

  • Bajo de peso: IMC < 18.5
  • Normal: 18.5 ≤ IMC < 24.9
  • Sobrepeso: 25 ≤ IMC < 29.9
  • Obeso: IMC ≥ 30

Valores para niños y adolescentes

El IMC para niños y adolescentes se evalúa en percentiles y se compara con gráficos de crecimiento. Se calcula de manera diferente según la edad y el sexo. Consulta a un profesional de la salud para interpretar los resultados en estos grupos de edad.
